Java source code for server side. Includes the whole java/eclipse project with code and the mysql driver used
to stablish connection to the database.

Files under /src/ contain source code
Files under /lib/ contain needed dependencies, libraries and drivers.

To install, copy the Betagaming_Server folder to your workspace and load it into eclipse.
Tested with:
-Eclipse Luna + Jboss (may not be necesary?)
