#This is the BetaGaming database documentation.
With this file you can find the last update of the MySql data files.

##INSTALATION
Copy all the folders and files to your /data/ folder in your mysql installation.
The "maindb1" database should now appear within your mysql database list.

##USAGE
This is a test database built to be used while the server core is being tested. Always
grab the last commit files to ensure that you use an updated scheme.
Tables included within this database:
-juegos
-partidas
-users
+relational databases

Names may change.
